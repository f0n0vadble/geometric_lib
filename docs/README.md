# Документация geometric_lib

## Описание
В данном проекте можно высчитать периметр и площадь трёх фигур: круг, квадрат и треугольник. Примеры использования можно увидеть ниже.

---

## Функционал `circle.py`

#### `area(r)`
Получает на вход радиус круга и возвращает его **площадь** по формуле `S = pi * r^2`.  

```python
area(5)  # 78.5398163397
```

---

#### `perimeter(r)`
Получает на вход радиус круга и возвращает его **периметр** по формуле `P = 2 * pi * r` 

```python
perimeter(9)  # 56.5486677646
```

---

## Функционал `square.py`

#### `area(a)`
Получает на вход длину стороны квадрата и возвращает его **площадь** по формуле `S = a^2`.  

```python
area(3)  # 9
```

---

#### `perimeter(a)`
Получает на вход длину стороны квадрата и возвращает его **периметр** по формуле `P = a * 4`

```python
perimeter(4)  # 16
```

---

## Функционал `triangle.py`

#### `area(a, h)`
Получает на вход основание и высоту треугольника и возвращает его **площадь** по формуле `S = 1/2 * a * h`.  

```python
area(12, 4)  # 24
```

---

#### `perimeter(a, b, c)`
Получает на вход длины сторон треугольника и возвращает его **периметр** по формуле `P = a + b + c`

```python
perimeter(2, 3, 4)  # 9
```

---

## Функционал `rectangle.py`

#### `area(a, b)`
Получает на вход длины сторон прямоугольника и возвращает его **площадь** по формуле `S = a * b` 

```python
area(12, 4)  # 48
```

---

#### `perimeter(a, b, c)`
Получает на вход длины сторон прямоугольника и возвращает его **периметр** по формуле `P = 2 * (a + b)`

```python
perimeter(2, 3)  # 10
```
